 == Description ==

WooFBPixel (FREE version) provides you with the ability to install your Facebook Pixel on your WooCommerce website in under 1 minute. Not only that, it allows you to track which specific products are being added to cart through custom Audiences in the Facebook Ad platform. This is possible due to our sku-append technology built right into the plugin.

Features:

- 1-Minute Facebook Pixel Setup: No more need to spend 20-30 minutes setting up the Facebook Pixel on your WooCommerce store. Single page plugin settings have you optimizing for Facebook Ads within 1 minute.

- Automatic Event Code Placement: Stop wasting time manually placing your event codes on cart, checkout, my account, and thank you pages. Our plugin does this automatically for you.

- Build Product-Specific Facebook Audiences: Track audiences who run carry specific products down your sales funnel. Build product-specific audiences based on your visitors‚Äô cart page traffic.

 == Installation ==
1. Download the plugin
2. Upload to "/wp-content/plugins" folder or install through WordPress Admin Dashboard
3. Activate the Plugin
4. Go to WooCommerce > WooFBPixel > Adjust settings
